*******
Read Me
*******

The *EducateWorkforce Guide for Users* is created using RST_ files and Sphinx_. You, the
user community, can help update and revise this documentation project on
GitHub::

  https://github.com/cucwd/ew-documentation

To suggest a revision, fork the project, make changes in your fork, and submit
a pull request back to the original project: this is known as the `GitHub Flow`_.
All pull requests need approval from EducateWorkforce. For more information, contact EducateWorkforce at `support@educateworkforce.com`_.
